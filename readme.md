# Currículo | Raul Falluh

Bem-vindo ao meu site de currículo! Aqui, você encontrará informações sobre minha formação, experiências e habilidades de maneira visual e interativa.

## 🖥️ Visão Geral

Este site foi desenvolvido com HTML, CSS e JavaScript, seguindo um design responsivo e moderno. O objetivo é apresentar meu currículo de forma atraente e acessível em qualquer dispositivo.

## 🎨 Design

O design do site foi inspirado na estética minimalista e moderna, com uma paleta de cores suave que se adapta ao tema claro ou escuro, dependendo da sua preferência. Os principais elementos do design incluem:

- **Tema Dinâmico:** Você pode utilizar o tema claro ou escuro de acordo com sua preferência.
- **Efeito de Rolagem Suave:** Navegação suave entre seções para uma melhor experiência do usuário.
- **Botão de Voltar ao Topo:** Facilita a navegação pela página.

## 📚 Seções do Currículo

1. **Sobre Mim:** Uma breve introdução e visão geral do meu background.
2. **Experiência:** Detalhes sobre minhas experiências anteriores.
3. **Habilidades:** Lista das principais habilidades que possuo, com ênfase em desenvolvimento web e programação.
4. **Formação:** Informações sobre minha educação, incluindo cursos e certificações.
5. **Portifólio:** Aqui você será redirecionado para a minha pagina do github, onde eu mantenho todos os meus projetos.
6. **Contato:** Um formulário de contato para que você possa entrar em contato comigo facilmente.

## 🔧 Tecnologias Utilizadas

- **HTML5:** Estrutura básica do site.
- **CSS3:** Estilização e layout responsivo.
- **JavaScript:** Funcionalidades interativas e dinâmicas.
- **Local Storage:** Armazenamento de preferências de tema.
